# Items CRUD API

Bienvenue sur la documentation de l'API Items.

## Introduction

Ce projet est une API REST robuste pour la gestion d'articles, construite avec **FastAPI**, **SQLModel**, et **PostgreSQL**. Il démontre une pipeline CI/CD professionnelle utilisant GitHub Actions, Semantic Release, et Docker.

## Fonctionnalités

- **CRUD Operations**: Créer, Lire, Mettre à jour, Supprimer des articles.
- **Base de données**: Intégration PostgreSQL via SQLModel.
- **CI/CD**: Tests automatisés, linting, sécurité, et gestion des releases.
- **Qualité**: Type checking (Mypy), Linting (Ruff), Sécurité (Bandit).
